# 💼 서비스 소개

```jsx
- 귀엽고 사랑스러운 내 반려동물! 누군가에게 자랑을 해보고 싶었던 적이 있으셨나요?
- 바로 지금, 반려동물을 자랑할 때 **펫피**를 이용해보세요 !
- 나만의 반려동물 카드를 만들고 마음껏 공유할 수 있어요
```

<br >
<br >

# 🗓️ 개발 기간

```jsx
24.03.10 ~ 24.03.14
```

<br >
<br >

# 💻 핵심 기능

```jsx
1. 아주 매력적인 반려동물 사진을 업로드해요.
2. 프레임을 고르면서 커스터마이징을 할 수 있어요.
3. 반려동물 정보를 입력하면 카드에 나타나요.
4. 완성된 카드를 다운로드 할 수 있어요.
```
